Aide mémoire des selecteurs déjà vus en css

  • * Selectionne tous les éléments de la page HTML
  • .ma-jolie-nav Selectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #-mon-unique-id Sélectionne tous les éléments ayant le nom de mon ID "mon-unique-id"
  • div, article, aside Sélectionne toutes les balises "div", "article" et "aside"
  • nav a Sélectionne les liens "a" inclus, imbriqués dans la balise "nav"

Propriétés CSS utilisées dans cette page

Decrivez leurs usages et leurs valeurs possibles

  1. Background: couleur de fond de l' élément
  2. Color: couleur de la police
  3. Float: effet flottant
  4. Font-family: police
  5. Font-size: taille de la police
  6. Height: hauteur de l' élément
  7. Left: position a partir de la gauche de la fenetre
  8. Line-height: hauteur de l' interligne
  9. Margin: marges extérieurs
  10. Padding: marges intérieurs
  11. Position: position de l' élément dans la page (on n' a utilisé que la position fixed pour l' instant)
  12. Text-align: alignement du texte
  13. Text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé
  14. Top: position a partir du haut de la fenetre
  15. Transition: temps de transition lors des animations (ici sur les liens de la barre de menu
  16. Width: largeur de l' élément par rapport a la taille de la fenetre si on indique une valeur relative (%, em etc...)
