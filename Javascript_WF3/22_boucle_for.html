Les boucles

Les boucles for et while, sont deux types de boucles, les plus tuilisées en JS. Servent a repeter plusieurs lignes de code

La boucle for

For prend 3 parametres encadrés dans ses parentheses, suivi entre accolades des instructions a executer autant de fois que la boucle l' indique. Les parametres sont

  • Initialisation: ici on declare la variable qui servira de compteur
  • Condition: Tant que cette condition est vraie cette boucle s' executera
  • expression finale: expression qui est executée a chaque fin de boucle
for(initialisation; condition; incrémentation){ instruction 1; instruction 2; }
