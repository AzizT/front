JQuery, une librairie Javascript pour gagner en productivité
"Write less, Do more" => Ecrire mois, faire plus

Version actuelle: V3.3.1

JQuery est une bibliotheque JS ( aussi appelé FrameWork ). C' est une boite a outils de blocs de code réutilisables
Elle est rapide, petite ( donc legere ) et riche en fonctionnalités. Elle simplifie considérablement la navigation et la manipulation dans les pages HTML
Il en est de même pour l' animation, gestion des évenements et AJAX. Tout cela est simplifié grace a une API ( application Programming Interface )
facile a utiliser et qui fonctionne sur beaucoup de navigateurs ( beaucoup de tests en permanences, meême sur des navigateurs anciens )
JQuery est elle même codée en JS.

En polyvalence et extensibilité, elle a changé la façon de travailler de millions de developpeurs.

  • Certaines fonctions sont toutes faites, déjà optimisées: Appel Ajax, boucles, animations, toggle...
  • Résoud les compatibilités navigateur. JQuery a des polyfills integrés
  • Lecture de code + simple, car les pbs de compatibilités sont gérés par la librairie

JQuery n' est pas une solution sur un code "sale" (Tag Soup)

JQuery a des millions de plug-ins additionnels

JQuery UI (User Interface), version actuelle: UI 1.12 (https://jqueryui.com)

C' est un ensemble d' interfaces utilisateurs, d' effets, de widgets ou de themes conçus a partir de la librairie JQuery

Si on a juste besoin d' un selecteur de dates dans un formulaire, JQuery UI est un choix idéal ( https://jqueryui.com/datepicker/=

JQuery Mobile Version 1.4.5

Est un framework d' interface utilisateur optimisé pourles ecrans tactiles

Charger JQuery dans une page web

La méthode la plus simple et basique, consiste a telecharger la librairie jquery.js sur le site officiel

Il faut prendre la version compressée, premier lien ( click droit sur ce lien et choisir dans le menu: " enregistrer la cible du lien sous", et mettre ce fichier dans un dossier js a la racine du site )

BIEN VERIFIER QUE LE FICHIER EST DANS LE BON DOSSIER JS

Le link est le suivant <script src="js/jquery-3.3.1.min.js"></script>
