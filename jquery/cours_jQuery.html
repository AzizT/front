facile a utiliser et qui fonctionne sur beaucoup de navigateurs ( beaucoup de tests en permanences, meême sur des navigateurs anciens )

selectionner le JQuery

Le coeur de JQ se fait dans la selection d' éléments pour ensuite executer des methodes dessus

$(selecteur) accepte en argument tout selecteur CSS, JQ nous renvoie ensuite une collection d' un ou plusieurs éléments DOM enveloppés dans un objet JQ

$('div') selectionne touts les divs

$('titre') selectionne tous les éléments ayant la classe "titre"

$('#boite') selectionne l' unique élément qui a l' ID "boite"

Cette enveloppe JQ est celle qui donne accés a toutes les fonctionnalités de JQ

  • Rajouter des fonctionnalités
  • Modifier le contenu HTML
  • Afficher ou masquer des éléments
  • Les animer etc...

Tousles éléments selectionnés en JS pur ( document.getElementById, querySelector) sont des éléments DOM et tous ceux selectionnés avec $('...') sont des éléments DOM dans une enveloppe JQ

JQuery dispose de plusieurs methodespour parcourir l' arbre du document ( le DOM )

  • chidren() parcourt les enfants directs
  • find() parcourt les descendants
  • next() renvoie au voisin suivant
  • prev() renvoie au voisin précédent
  • parent() renvoie au parent
  • eq() renvoie, parmi une liste d' éléments, celui qui se trouve a l' index fourni

Les Getters et les Setters

(Ajouter, retirer des classes et savoir si on a déjà des classes)
  • addClass(): ajoute une class
  • removeClass(): retire une class
  • toggleClass(): interrupteur qui ajoute ou retire une class
  • hasClass(): tester l' existence d' une class
  • is(): tester si on a plusieurs class

Les évenements

La plupart des interactions de l' utilisateur avec la page web peuvent etre interceptées, et une interaction lancée en retour

On peut dire a une fonction de s' executer lorsque l' utilisateur clique sur un bouton ou scrolle la fenetre ( et bien d' autres choses...appuie touche sur le clavier

  • mouse(souris):
    click: au clic sur l' élément
    mouseenter: lorsque la souris passe par dessus la zone de l' élément
    mouseleave: lorsque la souris sort de cette zone
  • keyboard ( le clavier):
    keydown: une touche de clavier a été enfoncée
    keyup; une touche de clavier a été relachée
  • Window ( la fenetre du navigateur )
    scroll: défilement de la fenetre
    resize: redimensionnement de la fenetre
  • Form ( le formulaire )
    change: pour les éléments <input>, <select> et <textarea> quand l' utilisateur change une de leur valeur
    focus et blur: lorsque la souris est dans l' input ou en sort
    submit: a l' envoi du formulaire
  • document (le contenu du DOM)
    DOMContentLoaded: lancé lorsque le document HTML est completement chargé et analysé sans attendre que les images et le CSS soient chargés
